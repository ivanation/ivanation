Visualizando Dataset Iris con Power BI y Modelo de Machine Learning

Como primer experimento, me gustaría combinar el poder de visualización de datos de Power BI con el potencial del machine learning. Para lograrlo, he seleccionado la conocida base de datos iris, la cual es comúnmente utilizada como el primer ejercicio de machine learning en la ciencia de datos, algo así como el "Hola mundo" en este campo.

Mi objetivo es crear un dashboard dinámico y fácil de utilizar en Power BI, mientras aprendo acerca de machine learning y redes neuronales. Este será el primero de varios sets de datos que abordaré, cada uno con mayor complejidad que el anterior. ¡Así que comencemos!

Visualizaciones

Para evitar sobrecargar la visualización con demasiadas gráficas, me concentré en la propiedad "petal length" al realizar la visualización de datos. Como se puede apreciar en la tabla de datos, esta propiedad es la más significativa a la hora de diferenciar los datos, ya que tiene una relación lineal con el tipo de iris.

En las visualizaciones se puede observar claramente los clúster o agrupaciones por clases, lo que indica que se trata de un modelo de clasificación en lugar de uno de regresión. Además, en este dataset, los clúster están claramente definidos, lo que facilitará el agrupamiento para los algoritmos de machine learning.

También agregué histogramas para visualizar la distribución de los datos a lo largo del eje x "petal length" y detectar si había alguna distribución que pudiera aportar más información para la clasificación de los datos, pero no se observó nada significativo.

Por último, incorporé un gráfico de radar para reflejar las diferencias en la propiedad "petal length" de cada clase de iris.

En conclusión, esta visualización ofrece una vista panorámica de los datos iris, haciendo hincapié en la propiedad "petal length", que representa muy bien la clasificación. Gracias a Power BI, fue muy fácil realizar este tipo de visualizaciones sin necesidad de escribir líneas de código, y se puede compartir como un reporte o insertarlo en un sitio web, como en nuestro caso.

Modelo Machine Learning

El siguiente paso en este ejercicio fue aplicar el machine learning.

Inicialmente, intenté elaborarlo en Python, pero no pude desplegar el modelo para que funcionara en la web. Aunque intenté utilizar Heroku, no pude lograrlo 😊. Espero poder hacerlo en la próxima oportunidad.

Para solucionar el problema de tener un modelo funcional, opté por utilizar TensorFlow.js, ya que funciona en JavaScript y vi una oportunidad para desplegar el modelo de manera más fácil. Y así fue.

No profundizaré en el uso de TensorFlow ni en la implementación del ejercicio con iris, ya que no es el alcance de esta publicación. En internet hay mucho material sobre el dataset iris y su implementación en diferentes lenguajes y plataformas.

En resumen, se utilizó una pequeña red neuronal de 10 neuronas (sigmoide) y 3 neuronas de salida (softmax), con un learning rate de 0.01 y 40 epochs, lo que resultó en una precisión del modelo del 99%.

El resultado fue una combinación de Power BI para las visualizaciones y el uso de TensorFlow para el modelo y la predicción de datos, totalmente funcional.
